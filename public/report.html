How to Report Malicious URLs

  • Start with Major Browsers

    Always report suspicious URLs to Google Safe Browsing and Microsoft Security Intelligence first. These services help protect millions of users worldwide and can quickly block access to malicious sites.

    Quick Action

    Use the links in the "Major Search Engines & Browsers" section below to report immediately.

  • Report to Infrastructure Providers

    Report the malicious URL to the services hosting or providing access to it:

    • 🏢Website's Hosting Provider
    • 🌐Domain's DNS Registrar
    • 🛡️CDN/Proxy Services
    How to Find Contacts

    Use these tools to find the right abuse contacts:

    • phish.report - Find hosting provider and domain registrar
  • Use Security Services

    Submit the URL to security services for analysis and broader protection:

    • 🔍URL Analysis (e.g., URLScan.io)
    • 🦠Malware Scanning (e.g., VirusTotal)
    • 📊Threat Intelligence
  • Report to Authorities

    For significant threats, report to your country's cybersecurity authorities.

    Region-Specific Reporting

    Scroll down to find the appropriate reporting channel for your region (US, EU, or other countries).
